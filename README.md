# REST API каталога организаций

Проект предоставляет REST API для работы с организациями, зданиями, офисами и видами деятельности.  
Реализованы фильтры и поиск по геолокации, видам деятельности и названию организации.
Согласно тестового, своевольно сделал возможность поддержки нескольких офисов у организации

---

## Запуск

### 1. Собрать контейнеры

```bash
docker compose build
```

### 2. Поднять инфраструктуру

```bash
docker compose up -d
```


### 3. Применить миграции

```bash
docker compose exec app poetry run alembic upgrade head
```


### 4. Засеять базу тестовыми данными

```bash
docker compose exec app poetry run python -m app.scripts.seed
```


После запуска API доступна по адресу:
http://localhost:8000/docs
